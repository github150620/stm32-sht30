# stm32-sht30
How to use SHT30 with STM32

## Pins
```
 STM32F104C8T6               SHT30
+-------------+            +-------+
|             |            |       |
|          PB10 <--------> SCL     |
|             |            |       |
|          PB11 <--------> SDA     |
|             |            |       |
+-------------+            +-------+
```
